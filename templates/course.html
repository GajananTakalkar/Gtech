Course Syllabus

1. PLC Programming (Allen-Bradley, Siemens, Delta, Mitsubishi)

  • Introduction to PLC
    • Basics of automation & control systems
    • Types & architecture of PLCs
    • PLC hardware components
    • I/O modules, power supply, communication ports
  • Programming Concepts
    • Ladder Logic, FBD, STL (Siemens)
    • Scan cycle and logic execution
    • Bit logic, Timers, Counters
    • Arithmetic & comparison operations
  • Hands-On Programming
    • Digital I/O applications
    • Interlocking & sequencing
    • Analog signal handling
    • Data handling (move, compare, scaling)
  • Advanced Topics
    • PID Control in PLC
    • Interrupts, subroutines
    • Troubleshooting & diagnostics
    • Communication with HMI/SCADA

2. SCADA System Development (Ignition, WinCC, Wonderware)

  • SCADA Basics
    • Introduction to SCADA & architecture
    • RTU vs PLC vs SCADA
    • Communication protocols (Modbus, OPC, Ethernet/IP)
  • SCADA Design
    • Tag creation and data binding
    • Real-time trends and historical trends
    • Alarm configuration
    • Event management and scripting
  • Hands-On Project
    • SCADA screen design
    • Integration with PLCs
    • User access management
    • Data logging and reporting

3. HMI Design (Allen-Bradley, Siemens, Delta)

  • Introduction to HMI
    • Difference between HMI & SCADA
    • Basic interface elements (buttons, sliders, displays)
  • Screen Development
    • Screen navigation
    • Real-time data display
    • Alarms, status indicators
    • Animation & graphics
  • Connectivity
    • Communication with PLC (RS232, Ethernet)
    • Address mapping & tag linking
    • Downloading & testing projects

4. VFD Setup & Control (ABB, Delta, Siemens)

  • Basics of Drives
    • What is a VFD and how it works
    • Applications of VFDs in industries
  • Configuration & Parameters
    • Drive wiring and control circuits
    • Speed, torque, acceleration setup
    • Fault diagnostics and safety features
  • PLC & VFD Interface
    • Analog/Digital control from PLC
    • Modbus or Ethernet-based control
    • Feedback & PID-based speed control

5. PLC with Python Integration

  • Python for Automation
    • Python Basics (variables, loops, functions)
    • Libraries: pyModbusTCP, snap7, pycomm3
  • PLC Communication
    • Reading/Writing data from PLC using Python
    • Real-time data acquisition
    • Logging and visualization
  • Automation Use-Cases
    • Email/SMS alerts on alarms
    • Data logging to Excel/CSV
    • Dashboard creation using Flask or Tkinter
    • REST API integration for remote monitoring
